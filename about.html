Notre métier, c'est la technologie pour vos communautés en ligne.

Nous pensons que les usages nés avec les réseaux sociaux vont s'étendre dans tous les domaines où la communication est nécessaire, et nous souhaitons fournir l'outil pour accompagner cette transformation.

Depuis 2011 nous produisons des applications pour équiper les entreprises, les collectivités et les associations dans leur communication et leur organisation interne. En 2014 nous créons la première solution open source qui permet de rendre sociales toutes les applications web (sites Internet, Intranet, applications métier, etc.)
